Profile

Objectives

This research work is going to focus on the following area Get headcount of chicks by single capture from the camera module Auto-regularise the room temperature for the chicks, by sensing the temperature change, and regulate it using the heater system Determine the mortality rate of chicks day in day out. Detect movement of chicks and send an alert message if no movement is observed consecutively for a specified time period.

Requirements

Software Requirement Raspbian OS: Linux operating system that runs on Raspberry PI. Python 2.7: For establishing communication with sensors. OpenCV lib: For image processing

Technical Details

Backend Development: Python for communicating with the sensors API development: This API section of the project would be developed with Java using Jersey. Frontend: HTML, CSS, Javascript, Angular JS Report/Analytics: HighCharts for interactive visualization
